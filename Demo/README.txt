Instructions on how to create and populate database:

1. Run player_bio_info_etl.py
2. Run player_stats_etl.py
3. Run player_shoes.py
4. Run salary.py

Then select * from each database to see ALL information.
As for seeing different queries follow instructions below:

1. Run menu.py
2. Follow instructions within menu.py


List of libraries needed to be installed:

prettytable
tabulate
Request
json
pymysql
beautifulsoul
urlopen

Included in the zip is a database export so all the information can be viewed at anytime.
